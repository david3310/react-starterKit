# react-kit
React Starter Kit for Quick Start on React !

# Demo


# Install and work
```
npm install
grunt start-dev-server

```

Now Open

[http://localhost:8000/component/Counter](http://localhost:8000/component/Counter)
or
[http://localhost:8000](http://localhost:8000)
